# Residenz-Life Installer (Prism/MultiMC)

Lade die `Residenz-Life-Installer.zip` aus den Releases herunter, ziehe sie in deinen Prism Launcher und starte das Spiel. Updates werden automatisch geladen.

Das Auto-Update wird ueber `packwiz-installer-bootstrap.jar` und den `preLaunchCommand` in `instance.cfg` ausgefuehrt.
